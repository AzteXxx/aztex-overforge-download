# AzteX Overforge

Otimizador para `Windows 10/11` com foco em desempenho real, mais responsividade, menos stutter e melhor consistencia em jogos.

## Download oficial

O download oficial do `AzteX Overforge` fica na aba `Releases` deste repositorio.

Arquivos da release:

- `AzteX-Overforge-v2.9.0.exe`
- `AzteX-Overforge-v2.9.0.zip`
- `SHA256SUMS.txt`

## Como usar

1. baixe a release
2. se baixar o `.zip`, extraia tudo
3. abra `AzteX Overforge.exe` como administrador
4. use a versao `Free`
5. se quiser desbloquear a `Pro`, copie o `machine hash` dentro da suite e solicite sua ativacao

## Free e Pro

### Free

- caminho publico padrao
- recursos base de otimizacao
- leitura e ajuste inicial do sistema

### Pro

- desbloqueio por `activation code`
- recursos premium de otimizacao
- packs mais fortes
- extras de desempenho avancado

## Integridade

Sempre confira o arquivo `SHA256SUMS.txt` da release para validar o pacote oficial.
